# 인체 생리 시뮬레이터 — 광고 vs 실제

미용/다이어트/수면 보조제가 광고에서 말하는 것과, 실제로 몸속(위·소장·간·뇌·피부·신장)에서
일어나는 일을 **좌우로 나란히** 비교하는 정적 웹앱입니다. 빌드 과정이 필요 없고,
`index.html`을 열기만 하면 바로 동작합니다.

## 실행 방법

**로컬에서 바로 열기**
`index.html`을 더블클릭해서 브라우저로 열면 됩니다.

**GitHub Pages로 배포하기**
1. 이 저장소를 GitHub에 push
2. Settings → Pages → Branch를 `main`, 폴더를 `/ (root)`로 설정
3. 몇 분 후 `https://<사용자명>.github.io/<저장소명>/` 에서 접속 가능

## 데이터베이스처럼 동작합니다

이 앱은 브라우저의 localStorage를 작은 데이터베이스처럼 씁니다 (`js/db.js`).

- 처음 실행하면 `js/devices.js`의 시드 데이터(글루타치온 필름 / 슬리피솔 / 다이어트 보조제) 3개로 DB가 채워집니다.
- 그 다음부터는 **화면에서 추가·수정·삭제한 내용이 유일한 데이터**가 됩니다. 새로고침하거나
  브라우저를 껐다 켜도 그대로 남아있습니다.
- 처음 3개 기기도 포함해서 **모든 항목을 자유롭게 수정하거나 삭제**할 수 있습니다 (`✎` / `✕`).
- 실수로 다 헝클어졌다면 왼쪽 아래 **"초기 데이터로 되돌리기"**로 리셋할 수 있습니다.
- `js/devices.js`는 딱 한 번, 최초 실행 시에만 쓰이는 시드용 파일입니다. 이미 DB가
  만들어진 브라우저에는 이 파일을 고쳐도 반영되지 않습니다.

## 기능

- **좌우 분할 비교**: 기기를 클릭하면 오른쪽에 광고 주장(왼쪽 컬럼)과 실제 기전(오른쪽 컬럼)이
  나란히 뜨고, 아래에 한계·부작용이 따로 정리됩니다.
- **인체 도해 연동**: 선택한 기기가 실제로 작용하는 기관이 인체 도해에서 확대·발광되고,
  파동/기포/수용체 결합 같은 애니메이션이 재생됩니다.
- **추가 / 수정 / 삭제**: `+ 새 기기 추가`로 새 항목을 만들고, 목록의 `✎`로 언제든 내용을
  고치고, `✕`로 지울 수 있습니다.
- **내보내기 / 불러오기**: 지금 이 브라우저의 DB 전체를 JSON으로 내보내 친구에게 보내거나,
  친구가 보낸 JSON을 불러와 합칠 수 있습니다. (id가 겹치면 새 항목으로 추가돼요.)
- **초기화**: 언제든 시드 데이터 3개로 되돌릴 수 있습니다.

## 모두의 화면에 항상 보이는 기본 항목을 바꾸고 싶다면

`js/devices.js`의 `SEED_DEVICES` 배열을 고치고 git commit & push 하세요.
(단, 이미 방문한 적 있는 브라우저는 이미 자기 DB가 있어서 시드가 다시 적용되지 않습니다 —
그 경우는 그 브라우저에서 "초기 데이터로 되돌리기"를 눌러야 새 시드를 받습니다.)

## 폴더 구조

```
body-ad-mythbuster/
├── index.html        메인 페이지
├── css/style.css      디자인 (다크 톤 + 좌우 분할 비교 레이아웃)
└── js/
    ├── devices.js      최초 1회용 시드 데이터 (SEED_DEVICES)
    ├── db.js           localStorage 기반 CRUD "데이터베이스" 계층
    └── app.js          화면 로직
```
